1. This sample code is for the customer who want to change the calibration data inside the camera.
2. The customer need build their own calibration tool to generate the new calibration data. 
3. Replace the new calibration value in the .c file.
4. compile the eeprom_flash_max96712_relese.c using "gcc eeprom_flash_max96712_relese.c"
5. ./a.out can download the new calibration data to the camera.
